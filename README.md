## ローカルでの構築・確認方法

### HugoのCLIツールを入れる

#### Mac
[Homebrew](https://brew.sh/index_ja)が必要

```sh
brew install hugo
```

#### Windows
[Chocolatey](https://www.cresco.co.jp/blog/entry/2127/)が必要

```posh
choco install hugo -confirm
```

#### Linux
パッケージマネージャがないので自前でビルドするかバイナリを落としてくるか(Golangが必要)

### Hugoのローカルサイトを起動

```sh
hugo server
```
